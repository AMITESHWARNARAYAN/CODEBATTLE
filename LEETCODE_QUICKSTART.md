# LeetCode-Style CodeEditor - Quick Start Guide

## 🚀 Getting Started

### 1. **Access the Code Editor**

Navigate to any problem in CodeBattle:
- Solo Practice mode
- Matchmaking mode
- Friend Challenge mode

The new LeetCode-style interface will load automatically.

## 🎯 Common Tasks

### Reading the Problem

1. **View Problem Description**
   - Default tab shows full problem
   - Read title, description, examples
   - Check constraints

2. **View Examples**
   - Scroll down to Examples section
   - See Input, Output, Explanation
   - Multiple examples provided

3. **Check Constraints**
   - Scroll to Constraints section
   - Understand problem limits

### Writing Code

1. **Select Language**
   - Click language dropdown
   - Choose: JavaScript, Python, Java, C++
   - Editor updates automatically

2. **Write Your Solution**
   - Type code in Monaco Editor
   - Syntax highlighting enabled
   - Line numbers visible
   - Auto-formatting available

3. **Test Your Code**
   - Click "Submit" button
   - View test results
   - See pass/fail status

### Getting AI Help (Solo Practice)

1. **Get Explanation**
   - Click "Get Explanation" button
   - AI generates problem overview
   - Appears in left panel

2. **Get Guidance**
   - Write some code first
   - Click "Get Guidance" button
   - AI provides feedback on your code

3. **View Full Solution**
   - Click "View Full Solution" button
   - AI generates complete solution
   - Learn from the approach

### Resizing Panels

1. **Adjust Panel Width**
   - Hover over divider between panels
   - Cursor changes to resize icon
   - Click and drag left/right
   - Release to set new width

2. **Constraints**
   - Left panel: 20% - 70% width
   - Right panel: 30% - 80% width
   - Smooth resizing animation

### Switching Tabs

1. **Description Tab**
   - Default tab
   - Problem details
   - Examples and constraints

2. **Editorial Tab**
   - Coming soon
   - Will show official solution

3. **Solutions Tab**
   - Coming soon
   - Community solutions

4. **Submissions Tab**
   - Coming soon
   - Your submission history

### Theme Toggle

1. **Dark Mode**
   - Click moon icon
   - Dark background
   - Light text
   - Better for night coding

2. **Light Mode**
   - Click sun icon
   - Light background
   - Dark text
   - Better for day coding

## ⏱️ Timer Management

### Timer Display
- Located in top right
- Shows MM:SS format
- Red color when < 60 seconds

### Auto-Submit
- When timer reaches 0
- Code auto-submits
- Toast notification shown
- Redirects to results

### Time Management Tips
- 10 minutes per match
- Plan your approach first
- Write code efficiently
- Submit before time runs out

## 🐛 Troubleshooting

### Panel Won't Resize
- Ensure cursor is on divider
- Cursor should show resize icon
- Try dragging again

### Code Not Saving
- Code auto-saves as you type
- Check browser console for errors
- Refresh page if needed

### AI Features Not Working
- Check Groq API key in backend
- Ensure backend is running
- Check browser console for errors

### Timer Not Counting Down
- Refresh the page
- Check browser console
- Restart the match

## ⌨️ Keyboard Shortcuts

### Monaco Editor
- **Ctrl+S**: Save (auto-saved)
- **Ctrl+Z**: Undo
- **Ctrl+Y**: Redo
- **Ctrl+/**: Toggle comment
- **Alt+Shift+F**: Format code
- **Ctrl+F**: Find
- **Ctrl+H**: Find and replace
